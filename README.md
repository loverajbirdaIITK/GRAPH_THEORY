# Graph-Theory and Route Optimization

Overview

This project models transportation networks as weighted graphs and applies classical graph algorithms to analyze connectivity, optimize routes, and evaluate network performance. It demonstrates how graph theory can be used to solve real-world transportation and routing problems by representing stations as vertices and routes as weighted edges.

The project implements shortest-path, graph traversal, and connectivity algorithms to identify efficient routes, analyze network structure, and evaluate alternative paths under different connectivity and distance constraints.

Features

* Modeled transportation networks using weighted graph representations.
* Implemented **Dijkstra's** algorithm for shortest-path computation.
* Implemented **Bellman-Ford** algorithm for handling weighted shortest-path problems.
* Applied **BFS** and **DFS** for graph traversal and connectivity analysis.
* Used **Tarjan's** and **Kosaraju's** algorithms to identify connected and strongly connected components.
* Evaluated route efficiency and network robustness under varying connectivity constraints.

Technologies Used

* **Language:** C++
* **Concepts:** Graph Theory, Weighted Graphs, Shortest Paths, Graph Traversal, Connected Components
* **Algorithms:** Dijkstra, Bellman-Ford, BFS, DFS, Tarjan, Kosaraju

Learning Outcomes

* Gained practical experience with graph data structures and algorithm implementation.
* Understood the trade-offs between different shortest-path algorithms.
* Applied graph-theoretic techniques to solve transportation and route-planning problems.
* Strengthened problem-solving skills through the implementation of fundamental graph algorithms.
